# coinz4s

This project attempts to create a common interface for interacting with multiple cryptocurrency wallets.

# Supported coins

- [x] BTC
- [x] LTC
- [ ] BCH
- [ ] DOGE
- [ ] DASH
- [ ] ZEC
- [ ] XMR
- [ ] EOS
- [ ] ETH
- [ ] USDT
- [ ] BNB
- [ ] ADA
- [ ] XLM
